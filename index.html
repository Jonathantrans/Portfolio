Millipede

Live Demo Â· Example Website

Millipede is a tool to quickly generate static sites from unsegmented text. It uses an LSTM to identify shifts in sentence topic along with a Jinja template to create a presentable and readable website. As opposed to other artificially intelligent website builders, such as Wix ADI, which typically provide a customized template for a user to fill with their own content, Millipede accepts pre-written content and adapts it to a single website design.
Millipede's source repository, which includes accreditation for the model used, is available on GitHub.

Wildfire Prediction

I envisioned this project as a complete machine learning application, from model training to deployment. Using a convolutional neural network with geographical data from national surveying programs such as the VIIRS Active Fire Product and the LANDFIRE Program, the probable spread of all active fires around the continental US is calculated and visualized on an interactive map.
The source for this project is not yet public. After the team that provided me with the dataset publishes their paper, I will be able to release the source code to the public.

Machine Learning Models in NumPy

After completing the excellent Stanford Machine Learning course by Andrew Ng on Coursera, I decided to implement some basic machine learning algorithms in pure NumPy. Each of the algorithms uses a real dataset (courtesy of the UCI Machine Learning Repository) and has visualizations using MatPlotLib and Seaborn.
The source code, which includes examples of linear regression, logistic regression, and a simple neural network, is hosted on GitHub.
